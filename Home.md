# SportBase Wiki — HKHK spordibaasi broneerimiskeskus

## Mis see projekt on?

HKHK tahab ehitada spordibaasi broneerimiskeskuse — süsteemi kus kogu kool saab broneerida treeningruume, registreeruda spordialadele ja jälgida ajakava.

Sina ehitad selle projekti raames kahte asja:
1. **Päriselt:** Linux server + MySQL andmebaas + backup süsteem
2. **Dokumendina:** kogu broneerimiskeskuse süsteemidisain — nagu päris pakkumine tellijale
